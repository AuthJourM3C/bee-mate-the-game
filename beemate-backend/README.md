# BeeMate Backend\n\nBackend proxy and game logic server for the BeeMate serious game.\n\n## Setup\n\n```bash\nnpm install\ncp .env.example .env\nnpm run dev\n```\n\n## API Endpoints\n\n| Method | Path | Description |\n|--------|------|-------------|\n| POST | /api/identify/image | Submit image for pollution identification |\n| POST | /api/identify/audio | Submit audio for pollution identification |\n| POST | /api/passcode/generate | Generate a passcode from game state |\n| POST | /api/passcode/restore | Restore game state from passcode |\n| GET | /api/facts/random | Get a random pollution fact |\n| GET | /health | Health check |\n\n## Environment Variables\n\n- `PORT` - Server port (default: 3001)\n- `CORS_ORIGIN` - Allowed CORS origin\n- `M3C_BASE_URL` - Base URL for M3C services\n- `MOCK_SERVICES` - Use mock responses (true/false)\n
